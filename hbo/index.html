HBO Treatment Automation System HiperSoft is a computer-based electronic control used in hyperbaric oxygen treatment systems. It is a state of the art automation system that provides the chamber's high comfortability and safe operation. With its enhanced profile entry and data logging features, HiperSoft is also flexible. A special algorithm is used to decrease the incidence of barotrauma, which is a common problem for patients undergoing hyperbaric therapy.
